[事前準備]
・Node.jsをインストール
https://nodejs.org/en
・Docker Desktopをインストール
https://www.docker.com/ja-jp/products/docker-desktop/
・MongoDB Compassをインストール
https://www.mongodb.com/ja-jp/products/tools/compass
・開発用にVSCode等のテキストエディタをインストール
https://code.visualstudio.com/

[手順]
1. dklike-api-server.zipを展開して任意のフォルダに置く

2. VSCodeでフォルダを開く

3. ターミナルでコマンド実行 (node_modulesのインストール)
npm install

4. ターミナルでコマンド実行 (Docker環境を構築)
docker-compose up -d

5. DockerでDBが起動していることを確認して、ターミナルでコマンド実行 (ローカルサーバーを起動する)
npm run dev

6. MongoDB Compassを起動してサーバーに接続する
接続先:localhost:27017
ユーザー名:root
パスワード:password

7. DBにデータを設定する
「db」という名前のデータベースに「contents」があるので、
documentsに用意してある「contents_sample.json」をインポート

8. アプリで動作テスト
タイトルメニューの設定内のカスタムサーバーにある「Base URL」にサーバーのアドレスを設定
(例) http://192.168.1.2:3000

設定を閉じた後、タイトルメニュー下の「コンテンツ」ボタンから
コンテンツ画面を開いて項目が表示されていれば成功です。
